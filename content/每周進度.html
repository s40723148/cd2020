新增網路設定

設定Port號(IPv4 & IPv6)

IP 查詢使用:

ifconfig

*若遇到Proxy問題

cd /etc/apt/apt.conf.d
sudo vi proxy.conf

#案I鍵進入編輯輸入以下內容:(編輯完成後按ESC>>:wq儲存編輯)
Acquire::http::proxy "http://[2001:288:6004:17::port號]:3128";

更新apt
sudo apt update

*ipv6環境設定

cd /etc/netplan

sudo vi 00-installer-config.yaml

#按I健進入編輯,ESC退出,:wq儲存
# This is the network config written by 'subiquity'
network:
    ethernets:
        enpes3:
            dhcp4: true
            dhcp6: true
            nameservers:
                addresses:
                    - 2001:b000:168::1
    version: 2

sudo netplan apply
